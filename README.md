#  Sistema de Agendamento para Barbearia

Um sistema simples de agendamento executado pelo terminal, para gerenciar horários de uma barbearia.

##  Funcionalidades

- Cadastro do nome do cliente
- Escolha de serviços
- Seleção de dia da semana (segunda a sábado)
- Escolha de horários disponíveis
- Remoção automática dos horários já agendados

##  Tecnologias Utilizadas

* Python 3
